# Security Policy

## Supported Versions

Currently, only the latest version of Dimensio is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

We take the security of Dimensio seriously. If you believe you have found a security vulnerability, please report it to us as follows:

1. **Email the Developer**: Send a detailed report to [Abdulkarim Shaikh](mailto:your-email@example.com) (Update with your actual email if preferred).
2. **Do not open a public issue**: Please allow us time to address the issue privately before it is disclosed publicly.

### What to include:
*   Description of the vulnerability.
*   The potential impact.
*   Steps to reproduce.
*   Your OS and environment details.
